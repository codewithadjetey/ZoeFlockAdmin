User Management

Manage system users and their access permissions:

User Roles and Permissions

Role Permissions Access Level
Super Admin Full system access, user management, settings Highest
Admin Most features, limited user management High
Pastor Member management, events, attendance, tithes Medium
Family Head Family members, limited attendance Limited
Member View own information, limited access Basic

Adding New Users

  1. Navigate to Users: Go to "Users" in the sidebar
  2. Add New User: Click "Add New User" button
  3. Fill User Details:
    • Full Name
    • Email Address
    • Username
    • Password
    • Role Assignment
  4. Set Permissions: Configure specific access rights
  5. Save User: Click "Save User" to create
Security Tips
  • Use strong passwords
  • Enable two-factor authentication
  • Regular password updates
  • Monitor user activity
  • Limit admin access
