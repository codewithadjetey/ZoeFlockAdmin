Recording Expenses

Record church expenses with detailed categorization:

Quick Access: Click "Expenses" in the Financial Management section of the sidebar.

Step-by-Step Expense Recording

  1. Navigate to Expenses: Click "Expenses" in the sidebar
  2. Add New Expense: Click "Add New Expense" button
  3. Enter Expense Details: Add description and amount
  4. Select Category: Choose expense category
  5. Set Date: Choose expense date
  6. Add Receipt: Upload receipt image
  7. Add Notes: Include any relevant notes
  8. Save Expense: Click "Save Expense" to record
Pro Tips
  • Record expenses promptly
  • Use consistent categories
  • Keep receipts organized
  • Add clear descriptions
