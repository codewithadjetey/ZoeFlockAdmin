Getting Started

To log in for the first time:

  1. Contact your system administrator to get your login credentials
  2. Navigate to the login page
  3. Enter your username and temporary password
  4. You'll be prompted to change your password on first login
  5. Set a strong password and confirm it

Note: If you don't have credentials, contact your church administrator.

After your first login, we recommend:

  1. Update your profile: Add your photo and contact information
  2. Review the dashboard: Familiarize yourself with the main features
  3. Add church members: Start by adding existing members to the system
  4. Set up events: Create regular service events
  5. Configure settings: Update church information and preferences

See our Quick Start Guide for detailed instructions.

To reset your password:

  1. Click "Forgot Password" on the login page
  2. Enter your email address
  3. Check your email for a reset link
  4. Click the link and enter a new password
  5. Confirm your new password

If you don't receive the email: Check your spam folder or contact your administrator.

Member Management

To add a new member:

  1. Go to "Members" in the sidebar
  2. Click "Add New Member"
  3. Fill in the required information:
    • First Name and Last Name
    • Date of Birth
    • Phone Number
    • Email Address
    • Address
  4. Add optional information (photo, emergency contacts)
  5. Click "Save Member"

Tip: Include a photo for easier identification during events.

Yes! You can import member data:

  1. Go to "Import/Export" in the sidebar
  2. Select "Members" tab
  3. Download the template file
  4. Fill in your member data
  5. Upload the completed file
  6. Review and confirm the import

Supported formats: CSV, Excel (.xlsx, .xls)

Note: Make sure your data matches the template format exactly.

To organize members into families:

  1. Go to "Families" in the sidebar
  2. Click "Add New Family"
  3. Enter family name and details
  4. Select a family head (primary contact)
  5. Add family members from existing member list
  6. Save the family

Benefits: Family organization helps with attendance tracking and communication.

Events and Attendance

To create a recurring Sunday service:

  1. Go to "Events" in the sidebar
  2. Click "Create New Event"
  3. Enter event details:
    • Name: "Sunday Service"
    • Category: "Sunday Services"
    • Date: Next Sunday
    • Time: Your service time
  4. Check "Recurring Event"
  5. Select "Weekly" frequency
  6. Choose "Sunday" as the day
  7. Set end date (optional)
  8. Save the event

Result: The system will automatically create Sunday services for the specified period.

The barcode system works as follows:

  1. Generate Barcodes: Each member gets a unique barcode
  2. Print ID Cards: Print member ID cards with barcodes
  3. Scan Attendance: Use mobile device to scan barcodes at events
  4. Automatic Recording: Attendance is recorded instantly

Benefits: Faster attendance recording, reduced errors, better data accuracy.

Setup: Contact your administrator to enable barcode functionality.

Yes, you can edit attendance records:

  1. Go to "Attendance" in the sidebar
  2. Find the event you want to edit
  3. Click on the event name
  4. Click "Edit Attendance"
  5. Make your changes
  6. Save the updated attendance

Note: Only users with appropriate permissions can edit attendance records.

Audit Trail: All changes are logged for accountability.

Financial Management

To record partial tithe payments:

  1. Go to "Tithes" in the sidebar
  2. Find the member's tithe record
  3. Click "Add Payment"
  4. Enter the payment amount
  5. Select payment method
  6. Add reference number (optional)
  7. Save the payment

Features: The system automatically tracks remaining balance and updates status.

History: All payments are recorded with timestamps and user information.

To generate financial reports:

  1. Go to "Reports" in the sidebar
  2. Select "Financial Reports"
  3. Choose report type:
    • Income Summary
    • Expense Breakdown
    • Tithe Analysis
    • Donation Summary
  4. Set date range
  5. Click "Generate Report"
  6. Download or print the report

Formats: PDF, Excel, CSV

Schedule: Set up automatic report generation.

Yes! You can export financial data:

  1. Go to "Import/Export" in the sidebar
  2. Select "Financial Data" tab
  3. Choose export format:
    • QuickBooks format
    • Excel spreadsheet
    • CSV file
  4. Set date range
  5. Select data categories
  6. Download the file

Compatibility: Works with most accounting software.

Security: Only authorized users can export financial data.

System and Technical

Zoe Flock Admin supports:

  • Chrome: Version 90+ (recommended)
  • Firefox: Version 88+
  • Safari: Version 14+
  • Edge: Version 90+

Mobile: Works on mobile browsers for basic functions.

Note: For best experience, use the latest browser version.

Data backup options:

  1. Automatic Backups: System performs daily backups
  2. Manual Export: Export data from Import/Export section
  3. Database Backup: Contact administrator for full backup

Backup Content: Members, events, attendance, financial records

Storage: Backups are stored securely in the cloud.

If you encounter an error:

  1. Note the error message: Write down the exact error
  2. Try refreshing: Refresh the page and try again
  3. Check your internet: Ensure stable internet connection
  4. Clear browser cache: Clear cookies and cache
  5. Contact support: If problem persists, contact your administrator

Error Reporting: Include error message, browser, and steps to reproduce.

Emergency: For critical issues, call your system administrator.
