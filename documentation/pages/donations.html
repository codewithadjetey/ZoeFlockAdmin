Recording Donations

Record donations from church members and visitors:

Quick Access: Click "Donations" in the Financial Management section of the sidebar.

Step-by-Step Donation Recording

  1. Navigate to Donations: Click "Donations" in the sidebar
  2. Add New Donation: Click "Add New Donation" button
  3. Enter Donor Info: Add donor name and contact details
  4. Set Amount: Enter donation amount
  5. Select Category: Choose donation category
  6. Choose Payment Method: Select payment type
  7. Add Notes: Include any relevant notes
  8. Save Donation: Click "Save Donation" to record
Pro Tips
  • Record donations promptly
  • Use consistent categories
  • Add clear donor notes
  • Generate receipts

Anonymous Donations

Support anonymous giving while maintaining records:

Anonymous Giving: Record donations without revealing donor identity.

Recording Anonymous Donations

  1. Select Anonymous: Check "Anonymous Donation" option
  2. Enter Amount: Record donation amount
  3. Choose Category: Select donation category
  4. Add Notes: Include any relevant details
  5. Save Record: Save without donor information

Anonymous Benefits

  • Privacy Protection: Respect donor privacy
  • Encourages Giving: Remove barriers to giving
  • Maintains Records: Keep financial records complete
  • Tax Compliance: Still generate receipts
