Understanding Attendance Tracking

Zoe Flock Admin provides comprehensive attendance tracking for all church events:

Attendance Features
  • Manual Recording: Mark attendance manually for each member
  • Barcode System: Scan member barcodes for quick check-in
  • Bulk Operations: Mark entire families or groups at once
  • Real-time Tracking: See attendance as it happens
  • Statistics: Generate detailed attendance reports
Benefits
  • Member Engagement: Track participation patterns
  • Event Planning: Understand event popularity
  • Follow-up: Identify members needing outreach
  • Reporting: Generate attendance statistics
  • Efficiency: Streamline check-in process

Accessing Attendance Management

Navigate to the attendance management section:

Quick Access: Click "Manage Attendance" in the sidebar navigation.
  1. Navigate to Attendance: Click "Manage Attendance" in the sidebar
  2. Select Event: Choose the event you want to manage
  3. Choose Method: Select manual or barcode recording
  4. Record Attendance: Mark members as present or absent
  5. Save Records: Confirm and save attendance data

Manual Attendance Recording

Record attendance manually for individual members:

Step-by-Step Manual Recording

  1. Select Event: Choose the event from the dropdown
  2. View Member List: See all registered members
  3. Mark Attendance: Click checkboxes for present members
  4. Add Notes: Include any relevant notes or comments
  5. Save Attendance: Click "Save Attendance" to record
Pro Tips
  • Use search to find members quickly
  • Mark families together for efficiency
  • Add notes for special circumstances
  • Review before saving

Barcode Attendance System

Use barcode scanning for quick and efficient attendance recording:

Barcode System: Scan member barcodes for instant attendance recording.

Setting Up Barcode Scanning

  1. Access Scanner: Go to "Scan Attendance Event"
  2. Select Event: Choose the event to record for
  3. Enable Camera: Allow camera access for scanning
  4. Scan Barcodes: Point camera at member barcodes
  5. Confirm Attendance: Verify member details
  6. Record Automatically: Attendance is recorded instantly

Barcode Benefits

  • Speed: Much faster than manual entry
  • Accuracy: Reduces human error
  • Efficiency: Streamlines check-in process
  • Real-time: Instant attendance recording
  • User-friendly: Easy for volunteers to use

Managing Attendance Records

Edit and manage existing attendance records:

Editing Attendance

  1. Navigate to the attendance management page
  2. Select the event and date
  3. Find the member record to edit
  4. Change attendance status (present/absent)
  5. Add or modify notes
  6. Save the updated record

Deleting Records

  • Admin Only: Only administrators can delete records
  • Confirmation Required: Confirm before deletion
  • Audit Trail: Deletions are logged for security
  • Data Recovery: Contact support for data recovery
