Adding New Members

To add a new member to your church database:

  1. Navigate to the Members section in the main dashboard
  2. Click the "Add New Member" button
  3. Fill in the required information:
    • Personal details (name, date of birth, contact information)
    • Family information (if applicable)
    • Emergency contact details
    • Membership date
  4. Click "Save Member" to add them to the database
Pro Tip

Always include a photo of the member for easier identification during events and attendance tracking.

Search and Filter

Find members quickly using the search and filter options:

Search Options
  • Name Search: Find members by first or last name
  • Phone Search: Search by phone number
  • Email Search: Find by email address
  • ID Search: Search by member ID
Filter Options
  • Age Groups: Filter by age ranges
  • Membership Status: Active, inactive, new, former
  • Family: Filter by family units
  • Date Range: Filter by join date
Important: Always ensure member data is kept secure and confidential. Only authorized users should have access to member information.
