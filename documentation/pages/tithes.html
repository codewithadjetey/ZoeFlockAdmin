Creating Tithe Records

Set up tithe commitments for church members:

Quick Access: Click "Tithes" in the Financial Management section of the sidebar.

Step-by-Step Tithe Creation

  1. Navigate to Tithes: Click "Tithes" in the sidebar
  2. Add New Tithe: Click "Add New Tithe" button
  3. Select Member: Choose the member from the dropdown
  4. Enter Amount: Set the tithe amount
  5. Set Frequency: Choose payment frequency (monthly, weekly, etc.)
  6. Set Start Date: Choose when tithe begins
  7. Add Notes: Include any relevant notes
  8. Save Tithe: Click "Save Tithe" to create
Pro Tips
  • Set realistic amounts
  • Use consistent frequencies
  • Add clear notes
  • Review before saving

Partial Payment System

Record partial tithe payments with the advanced payment system:

Partial Payments: Record multiple payments toward a single tithe commitment.

Recording Partial Payments

  1. Find Tithe Record: Locate the member's tithe record
  2. Click "Record Payment": Select the payment option
  3. Enter Payment Amount: Specify the payment amount
  4. Select Payment Method: Choose payment type
  5. Add Payment Notes: Include payment details
  6. Save Payment: Record the partial payment

Payment Methods

  • Cash: Physical cash payments
  • Check: Check payments with reference numbers
  • Bank Transfer: Electronic bank transfers
  • Mobile Money: Mobile payment services
  • Online Payment: Digital payment platforms

Marking Tithes as Paid

Complete tithe payments and mark as fully paid:

Mark as Paid: Use the "Mark as Paid" feature to complete tithe payments.

Marking Full Payment

  1. Find Tithe Record: Locate the member's tithe
  2. Click "Mark as Paid": Select the mark as paid option
  3. Confirm Payment: Verify the remaining amount
  4. Add Notes: Include payment completion notes
  5. Complete Payment: Mark tithe as fully paid

Payment Completion

  • Status Update: Tithe status changes to "Paid"
  • Balance Zero: Remaining amount becomes zero
  • Completion Date: Record completion date
  • Payment Summary: Generate payment summary

Managing Tithe Records

Edit and manage existing tithe records:

Editing Tithe Records

  1. Navigate to the tithes management page
  2. Find the tithe record to edit
  3. Click "Edit" or "Modify" button
  4. Update tithe details (amount, frequency, etc.)
  5. Add or modify notes
  6. Save the updated record

Deleting Tithe Records

  • Admin Only: Only administrators can delete records
  • Confirmation Required: Confirm before deletion
  • Payment History: Consider payment history before deleting
  • Audit Trail: Deletions are logged for security
