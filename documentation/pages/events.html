Creating Events

Events are the foundation of attendance tracking. Here's how to create and manage them:

Step-by-Step Event Creation

  1. Navigate to Events: Click "Events" in the sidebar
  2. Create New Event: Click "Create New Event" button
  3. Fill Event Details:
    • Event Name: Descriptive name (e.g., "Sunday Service")
    • Event Category: Select appropriate category
    • Date & Time: Set start and end times
    • Location: Event venue or location
    • Description: Additional details about the event
    • Recurring: Set if event repeats regularly
  4. Save Event: Click "Save Event" to create
Pro Tips
  • Use clear, descriptive event names
  • Set up recurring events for regular services
  • Include location details for members
  • Add descriptions for special events

Recording Attendance

Track member attendance for each event:

Manual Attendance Recording

  1. Select Event: Go to "Attendance" and choose the event
  2. Record Attendance: Click "Record Attendance"
  3. Mark Attendance: Check boxes for present members
  4. Add Notes: Optional notes for absences
  5. Save: Click "Save Attendance"

Barcode Attendance System

Advanced Feature: Use barcode scanning for quick attendance recording.
  1. Generate Barcodes: Create barcodes for members
  2. Scan Attendance: Use mobile device to scan member barcodes
  3. Automatic Recording: Attendance is recorded automatically
Attendance Tips
  • Record attendance promptly
  • Use barcode system for large events
  • Add notes for absences
  • Review attendance regularly

Managing Events

Edit, delete, and manage existing events:

Editing Events

  • Click on the event name to edit
  • Modify event details as needed
  • Update recurring event settings
  • Save changes

Deleting Events

  • Select the event to delete
  • Confirm deletion action
  • Note: This also deletes attendance records
  • Use with caution

Recurring Events

Set up events that repeat regularly:

Recurring Event Options
Frequency Options:
  • Daily: Events that occur every day
  • Weekly: Events that occur weekly (e.g., Sunday services)
  • Monthly: Events that occur monthly
  • Custom: Specific recurring patterns
Benefits:
  • Automatic event creation
  • Consistent attendance tracking
  • Reduced manual work
  • Better organization

Best Practices

Do's
  • Create events well in advance
  • Use descriptive event names
  • Record attendance promptly
  • Review attendance reports regularly
  • Set up recurring events for regular services
Don'ts
  • Don't delete events with attendance records
  • Don't use vague event names
  • Don't delay attendance recording
  • Don't ignore attendance patterns
  • Don't forget to set event categories
Important: Regular attendance tracking helps identify members who may need pastoral care or follow-up.
