  • Leadership: Assign group leaders and roles
  • Reporting: Generate group-specific reports

Creating Groups

Follow these steps to create and manage groups in your church:

Step-by-Step Group Creation

  1. Navigate to Groups: Click "Groups" in the sidebar navigation
  2. Create New Group: Click the "Add New Group" button
  3. Enter Group Details:
    • Group Name: Enter a descriptive group name
    • Description: Add group purpose and activities
    • Category: Select group type (ministry, committee, etc.)
    • Meeting Schedule: Set regular meeting times
    • Location: Specify meeting location
  4. Assign Leadership: Select group leader and co-leaders
  5. Add Members: Select members to join the group
  6. Set Permissions: Configure group access levels
  7. Save Group: Click "Save Group" to create
Pro Tips
  • Use clear, descriptive names
  • Assign capable leaders
  • Set realistic meeting schedules
  • Include group descriptions

Managing Group Members

Add, remove, and manage group membership:

Adding Members to Groups

  1. Go to the group details page
  2. Click "Add Member" or "Manage Members"
  3. Search for existing members
  4. Select members to add
  5. Assign roles (leader, co-leader, member)
  6. Set join date and notes
  7. Save member assignments

Removing Group Members

  1. Navigate to group member management
  2. Find the member to remove
  3. Click "Remove" or "Leave Group"
  4. Confirm the removal action
  5. Update group statistics

Group Events and Activities

Organize and track group events:

Event Management: Create and manage group-specific events and activities.

Creating Group Events

  1. Go to group details page
  2. Click "Add Event" or "Schedule Meeting"
  3. Enter event details (name, date, time, location)
  4. Set event type (meeting, activity, training)
  5. Add event description and notes
  6. Invite group members
  7. Save the event

Event Management

  • Recurring Events: Set up regular meetings
  • Event Reminders: Send notifications before events
  • Attendance Tracking: Record who attended
  • Event Notes: Share meeting minutes and updates

Importing Group Data

Import group information from external sources:

Import Feature: Use the Import/Export section to bulk import group data from CSV or Excel files.

  2. Download Template: Get the import template from Import/Export section
  3. Fill Template: Add your group data to the template
  4. Upload File: Upload the completed template
  5. Review Data: Check the imported data for accuracy
  6. Confirm Import: Complete the import process

Best Practices

Group Management Best Practices
