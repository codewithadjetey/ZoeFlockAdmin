Getting Started in 5 Steps

Step 1: Access the System

  1. Open your web browser
  2. Navigate to your Zoe Flock Admin URL
  3. Enter your login credentials
  4. Click "Sign In"
Note: If you don't have login credentials, contact your system administrator.

Step 2: Add Your First Member

  1. Click on "Members" in the sidebar
  2. Click "Add New Member" button
  3. Fill in the required information:
    • First Name and Last Name
    • Date of Birth
    • Phone Number
    • Email Address
    • Address
  4. Click "Save Member"

Step 3: Create an Event

  1. Go to "Events" in the sidebar
  2. Click "Create New Event"
  3. Enter event details:
    • Event Name
    • Date and Time
    • Location
    • Description
  4. Click "Save Event"

Step 4: Record Attendance

  1. Go to "Attendance" in the sidebar
  2. Select the event you created
  3. Click "Record Attendance"
  4. Check off members who attended
  5. Click "Save Attendance"

Step 5: Record Tithes

  1. Navigate to "Tithes" in the sidebar
  2. Click "Add New Tithe"
  3. Select the member
  4. Enter the tithe amount
  5. Choose payment method
  6. Click "Save Tithe"
Quick Tips
  • Start with adding members
  • Create events regularly
  • Record attendance promptly
  • Keep financial records updated
  • Use the search function

System Overview

Zoe Flock Admin is a comprehensive church management system with the following main features:

Member Management

Add, edit, and manage church members with detailed profiles.

Events & Attendance

Create events and track member attendance automatically.

Financial Management

Track tithes, donations, expenses, and generate reports.

Reports & Analytics

Generate detailed reports and view attendance statistics.
